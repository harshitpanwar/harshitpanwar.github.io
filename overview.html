C++ is a general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language, or "C with Classes".

By the word "general" we mean that it has applications in various fields. As mentioned above it is an extension to the pre-existing C language. C++ is an Object Oriented Programming language meaning that it involves classes and objects.We will discuss this in the upcoming topics. The main difference between C and C++ is Object Orientation only . As there is not much difference between these languages sometimes they are also referred simultaneously.

C++ is still among the fastest languages in the world as it is more closer to the hardware components as compared to any other language.
